Simulador de Autómatas y Rutas en Tablero

Aplicación web interactiva que simula movimiento de múltiples autómatas en cuadrícula 4x4. Implementa algoritmo DFS (Depth First Search) para calcular rutas que cumplan patrones de restricción de colores. Incluye visualización dinámica de grafos NFA (Autómata Finito No Determinista) con SVG, gestión de concurrencia con sistema de turnos, detección de colisiones, y análisis de rutas ganadoras vs. totales generadas.

JavaScript ES6+ HTML5 CSS3 SVG DFS Algorithm NFA Vanilla JS
